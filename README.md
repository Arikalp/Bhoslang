## 🎭 What Even Is This Madness?

Bhoslang is what happens when you combine:
- 🇮🇳 Indian street slang ("bsdk" = "behenchod")
- 💻 Programming language fundamentals
- 😂 Questionable naming conventions
- 🔥 A burning desire to make programming fun again

It's a beginner-friendly interpreted language that proves you don't need fancy syntax to build something awesome. Or at least... something.

## 🚀 Features That Will Make Your Computer Question Its Existence

### ✅ **Currently Working (Mostly)**
- **Variable Declaration** - `bsdk x = 42` (because variables need existential validation)
- **Variable Updates** - `badalbsdk x = x + 1` (change is the only constant)
- **Printing Values** - `likhbsdk x` (shout your variables to the world)
- **Math Operations** - `batabsdk x + y` (when calculator apps are too mainstream)
- **If/Else Statements** - `agarbsdk`/`nahitohbsdk` (when life gives you choices)
- **Comparison Operations** - `==`, `!=`, `>`, `<`, `>=`, `<=` (because sometimes you need to compare)
- **Help System** - `madadbsdk` (for when you forget everything)

### ❌ **Currently Broken (Read: Future Features)**
- Strings (who needs words when you have numbers?)
- Loops (circular logic is overrated)
- Functions (copy-paste is the original function)
- Error handling (errors are just unexpected features)
- Arrays (lists are for organized people)

### **The Three Pillars of Questionable Design**

1. **Tokenizer** - Splits your code into tokens like a ninja with ADHD
2. **Parser** - Creates an Abstract Syntax Tree (sounds fancy, isn't)
3. **Interpreter** - Actually runs your code (the magic part)
