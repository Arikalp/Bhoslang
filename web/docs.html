🚀 Bhoslang — Official Documentation

Welcome to Bhoslang — a lightweight, beginner-friendly programming language designed to make learning interpreters fun.

If you know basic logic and can type without fear… Congratulations — you are already overqualified. 🎉

🧠 What Bhoslang Can Do (Phase 1)

✅ Creating variables
✅ Updating variables
✅ Printing values
✅ Performing math (+ − * /)
✅ Looking suspiciously professional 😎

📜 Ground Rules (Read or Regret)

🔹 Write one command per line
🔹 Spaces between tokens are REQUIRED (this is not Python magic)
🔹 Keywords are case-sensitive
🔹 Declare variables before using them (trust me on this)

🔥 COMMANDS REFERENCE

🟢 bsdk — Create a Variable

Use this when you want the computer to remember something.

Syntax: bsdk <variable> = <value>
Examples: 
bsdk x = 10
bsdk score = 99

🟡 badalbsdk — Update a Variable

Because change is the only constant (and bugs).

Syntax: badalbsdk <variable> = <expression>
Examples: 
badalbsdk x = x + 5
badalbsdk score = score * 2

🔵 likhbsdk — Print a Variable

The "just show me the value" command.

Syntax: likhbsdk <variable>
Example: 
bsdk x = 10
likhbsdk x
Output: 10

🟣 batabsdk — Print an Expression

When you don't trust your mental math.

Syntax: batabsdk <value> <operator> <value>
Examples: 
batabsdk 10 + 5
batabsdk x * y
batabsdk 20 / 5

🔥 agarbsdk — If Statement

When life gives you choices, make decisions programmatically.

Syntax: agarbsdk <condition>
Examples: 
agarbsdk x == 10
likhbsdk x
nahitohbsdk
likhbsdk 0

🔥 nahitohbsdk — Else Statement

Plan B for when your if statement fails you.

Syntax: nahitohbsdk (follows agarbsdk)
Example: 
agarbsdk score > 50
likhbsdk score
nahitohbsdk
likhbsdk 0

🔥 madadbsdk — Show Help

When you forget everything and need guidance.

Syntax: madadbsdk

➕ Supported Operators

+ Addition x + y
- Subtraction x - y
* Multiplication x * y
/ Division x / y
== Equal to x == y
!= Not equal to x != y
> Greater than x > y
< Less than x < y
>= Greater than or equal x >= y
<= Less than or equal x <= y

🎬 Complete Example Program

Copy this into the terminal and feel powerful:
